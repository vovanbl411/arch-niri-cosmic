# Описание категорий пакетов

Этот документ описывает все категории пакетов в проекте PKGS-ARCH и объясняет, какие пакеты к какой категории относятся.

## Core (Ядро системы)

Пакеты в этой категории содержат базовые компоненты системы:

- `base.txt` - основные пакеты для функционирования системы, включая среду рабочего стола (в вашем случае - niri и связанные пакеты)
- `system.txt` - системные пакеты, включая ядро, прошивки, загрузчик, утилиты системы
- `network.txt` - пакеты для работы с сетью, включая iwd, openssh, openvpn и другие

## Desktop (Рабочий стол)

Пакеты, связанные с пользовательским интерфейсом и приложениями:

- `apps.txt` - пользовательские приложения, такие как браузеры, мессенджеры, офисные приложения
- `audio-video.txt` - пакеты для работы с аудио и видео, включая pipewire, кодеки, драйверы
- `greeter.txt` - средства входа в систему (в вашем случае - greetd и tuigreet)
- `niri.txt` - пакеты для среды рабочего стола Niri

## Development (Разработка)

Пакеты, полезные для разработчиков:

- `utils.txt` - утилиты командной строки и инструменты разработки (git, rust, go, nodejs, npm, bat, ripgrep, fd, fzf, zoxide, yazi, lsd, btop, htop, ncdu и др.)

## Fonts-Themes (Шрифты и темы)

Пакеты, отвечающие за внешний вид системы:

- `fonts.txt` - различные шрифты (включая Nerd Fonts) и темы оформления

## Hardware (Оборудование)

Пакеты, необходимые для работы с различным оборудованием:

- `drivers.txt` - драйверы устройств (в вашем случае - bluez для Bluetooth и др.)

## Virtualization (Виртуализация)

Пакеты для работы с виртуализацией:

- `virt.txt` - qemu, libvirt, virt-manager и другие инструменты виртуализации

## AUR (AUR пакеты)

Пакеты из репозитория AUR (Arch User Repository):

- `aur.txt` - сторонние пакеты, недоступные в официальных репозиториях
- `cosmic.txt` - пакеты для среды рабочего стола COSMIC (в AUR)

## Порядок установки

При установке пакетов рекомендуется следовать определенному порядку:

1. Core (особенно system.txt и base.txt)
2. Hardware
3. Virtualization
4. Desktop
5. Development
6. Fonts-Themes
7. AUR

Этот порядок помогает избежать конфликтов зависимостей.
